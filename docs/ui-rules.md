# Règles UI/UX

- Mobile first obligatoire sur tous les composants
- Design premium, pas d'interface générique
- Animations légères (pas d'effet lourd)
- Pas de popup inutile
- Responsive parfait mobile
- Cohérence visuelle entre tous les écrans
- Usage post-séance : utilisateur fatigué, luminosité faible → contraste suffisant
